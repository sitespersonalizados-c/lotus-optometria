# 👁️ Lótus Optometria - Sistema de Agendamento Online

Este é o site oficial de agendamento da **Lótus Optometria**, localizada em São José dos Campos - SP. O objetivo deste projeto é facilitar a marcação de exames de vista e consultas de rotina para os pacientes do optometrista **Fabiano Ricardo da Silva**.

## 🚀 Funcionalidades

* **Agendamento Simplificado**: O paciente escolhe o serviço e o período (Manhã ou Tarde).
* **Integração com WhatsApp**: Os dados do formulário são enviados diretamente para o WhatsApp do consultório: `(12) 98705-4594`.
* **Consulta de Valores**: Opção específica para pacientes que desejam saber preços e formas de pagamento antes de agendar.
* **Design Responsivo**: Otimizado para dispositivos móveis, garantindo que o paciente consiga agendar pelo celular.

## 🛠️ Tecnologias Utilizadas

* **HTML5**: Estrutura do site.
* **CSS3**: Estilização baseada na identidade visual da Lótus (Cores Turquesa e Azul Escuro).
* **JavaScript**: Lógica de captura de dados e integração com a API do WhatsApp.
